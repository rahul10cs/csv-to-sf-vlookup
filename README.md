## Overview
Stop doing manual VLOOKUPs against Salesforce records.

**CSV to SF VLOOKUP** is a lightweight, secure tool designed for Salesforce Professionals to enrich local CSV data with Salesforce data. Whether you need to pull "Account Owner" for a list of IDs, "Contact Email" for a lead list, or match Salesforce IDs to any field, this tool handles it in seconds.

## How to Use

1.  Open the extension while on any active Salesforce tab.
2.  Upload your CSV file.
3.  Map your CSV column (**"Lookup Value"**) to a Salesforce field (e.g., Email, Record ID, or External ID).
4.  Select the specific fields you want to retrieve from Salesforce.
5.  Click **"Run VLOOKUP"** to download your enriched CSV instantly.
